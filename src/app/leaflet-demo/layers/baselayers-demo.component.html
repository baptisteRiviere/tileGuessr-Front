Simple Base Layers

In this example, we create a map with multiple base layers using [leafletBaseLayers]. The [leafletBaseLayers] directive automatically adds the layers control so the user can switch between base layers. You can optionally provide options for the layers control using [leafletLayersControlOptions]. 

<div leaflet style="height: 300px;"
        [leafletOptions]="options"
        [leafletBaseLayers]="baseLayers"
        [leafletLayersControlOptions]="layersControlOptions">
</div>
